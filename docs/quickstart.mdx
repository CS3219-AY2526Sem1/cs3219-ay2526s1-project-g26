Before starting, make sure the following tools are installed:

### Git

- Download: [https://git-scm.com/downloads](https://git-scm.com/downloads)

### Docker

- Download: [https://www.docker.com/get-started](https://www.docker.com/get-started)

<Warning>
  Ensure Docker Desktop is **running** (especially for Windows/macOS users) as
  `dockerd` may not start without Docker Desktop
</Warning>

<Note>
  macOS M-chip users may use [OrbStack](https://orbstack.dev/) in replace of
  Docker Desktop
</Note>

<Note>
  Linux users may need to add themselves to the `docker` group to avoid using
  `sudo`
</Note>

Check if docker is installed:

```bash
docker --version
```

## Clone the Repository

Use Git to clone the project and move into the project directory:

```bash
git clone https://github.com/CS3219-AY2526Sem1/cs3219-ay2526s1-project-g26
cd cs3219-ay2526s1-project-g26
```

## Build with Docker

Navigate into `docker` folder:

```bash
cd docker
```

Start building with `docker compose`:

<Note>It may take some time for the initial build</Note>

<CodeGroup>
```bash docker-compose V2 (recommended)
docker compose -f docker-compose.dev.yml up --build -d
```

```bash docker-compose V1
docker-compose -f docker-compose.dev.yml up --build -d
```

</CodeGroup>

## Open the Project

1. You can access the `frontend` from any browser via `http://localhost`
2. You can access the `documentation` from any browser via `http://localhost:3005`
3. You can access the `user-service` endpoints via `http://localhost/api/user`
4. You can access the `question-service` endpoints via `http://localhost/api/question`
